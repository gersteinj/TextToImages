Text to Images

An updated version of my original Text to Images tool, this time as a web app so you don't need to install and run Python.

There is still work to be done on this, but for now you can save one image at a time. Your settings will not be deleted after a save, so you can make multiple images quickly. Scroll down to the bottom for more info if you're confused.

Settings

Basic Controls
Type in your word. Click save. This will open a save file dialog on your computer.
Text Color
Set the color of the text
BG Color
Set the color of the background
Choose a font
Type in the name of a font you want to use. For now, it's based on the fonts already installed on your computer. Safe defaults available on any system are: 'monospace', 'serif', and 'sans-serif'
Text Size
This is a slider to control the size of the text
